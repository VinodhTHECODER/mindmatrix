# 🚌 Smart Bus Booking Web Application

This project is a fully functional front-end bus ticket booking system developed using HTML, CSS, and JavaScript. It replicates the core features of modern booking platforms by providing an intuitive and interactive user experience.

---

## ✨ Key Highlights

- Real-time seat availability tracking
- 2+2 bus seat layout with aisle separation
- Dynamic seat selection with visual feedback
- End-to-end booking flow (Search → Select → Pay → Confirm)
- Passenger data collection with validation
- Payment interface simulation (UPI & Card)
- Automated receipt generation with complete journey details
- Ticket history management system
- Ticket cancellation with seat restoration

---

## 🎯 Objective

The goal of this project is to demonstrate how a real-world ticket booking system operates, focusing on:
- User interaction design
- Dynamic UI updates
- State management using JavaScript

---

## 🛠️ Tech Stack

- **HTML** – Structure  
- **CSS** – Styling and layout  
- **JavaScript** – Application logic and interactivity  

---

## ⚡ Unique Aspects

- Built entirely in a **single HTML file**
- No external libraries or frameworks used
- Efficient handling of seat states (available, selected, booked)
- Simple yet scalable architecture

---

## 🚧 Limitations

- No backend or database integration
- Data is stored temporarily (resets on refresh)
- Payment system is simulated

---

## 🔮 Future Scope

- Backend integration with database
- Real-time booking system
- Secure authentication
- Live payment gateway integration
